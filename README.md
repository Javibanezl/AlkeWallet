# Alke Wallet

Alke Wallet es una billetera digital simulada desarrollada como proyecto final para el módulo de "Fundamentos del Desarrollo Front-end". La aplicación permite a los usuarios gestionar un saldo virtual, realizar depósitos, transferir dinero a contactos y visualizar un historial de movimientos.

## Funcionalidades

El proyecto cuenta con las siguientes características funcionales:

- **Inicio de Sesión:** Pantalla de login con validación de credenciales.
- **Menú Principal:** Visualización del saldo actual y accesos rápidos.
- **Depósitos:** Simulación de carga de saldo con actualización automática.
- **Enviar Dinero:** - Gestión de contactos (Agregar nuevos contactos).
  - Realizar transferencias (descuento de saldo y registro de transacción).
- **Movimientos:** Historial dinámico que registra tanto los depósitos como las transferencias realizadas.
- **Persistencia de Datos:** Uso de `localStorage` para mantener el saldo, contactos y transacciones aunque se cierre el navegador.
